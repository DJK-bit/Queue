# Array circular queue Implementation

## 1. Project Description
This project enables the user to manually create and manipulate a queue. Furthermore, the project enables the user to have a deeper understanding of how a queue works without using Java's built in class.

## 2. Tech Stack/ Language
- Language: Java 
- IDE: Visual Studio Code

## 3. File Structure
- Main.java
- MyQueue.java
- README.md

## 4. Features
- enqueue(int value): Adds a new value to the rear of the queue.
- dequeue(): Removes and returns the value at the front of the queue.
- isEmpty(): Checks if the queue is empty.
- isFull(): Checks if the queue is full.
- peek(): Returns the value at the front of the queue.
- size(): Gets the size of the queue.
- capacity(): Gets the capacity of the queue.
- print(): Prints all elements inside of the queue.

## 5. How to Run
javac *.java
java Main

## 6. Time Complexity
- enqueue(int value): O(1)
- dequeue(): O(1)
- isEmpty(): O(1)
- isFull(): O(1)
- peek(): O(1)
- size(): O(1)
- capacity(): O(1)
- print(): O(n)
